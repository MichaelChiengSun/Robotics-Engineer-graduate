To run the package:

1) unzip the 210413294_cr_week8_test.zip folder in your catkin workspace;
2) build the catkin workspace;
3) in the first terminal, run "roscore";
4) run the human_robot_interaction.launch launch file;
5) in a new terminal, run "rostopic echo /objectInfo";
6) in a new terminal, run "rostopic echo /humanInfo";
7) in a new terminal, run "rostopic echo /perceivedInfo";
8) in a new terminal, run "rostopic echo /robotInfo".
